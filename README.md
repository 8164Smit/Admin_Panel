🚀 Full-Stack Admin Panel (Node.js + Express.js + MongoDB)

A modern, responsive, and fully functional Admin Panel built using Node.js, Express.js, MongoDB, Mongoose, EJS, Bootstrap 5, Passport.js, and MVC Architecture. This project demonstrates secure authentication, complete CRUD operations, database relationships using populate(), image uploads, email verification, and professional dashboard management.

✨ Features
🔐 Secure Admin Authentication (Login & Logout)
👤 Admin Profile Management
🔑 Change Password Functionality
📧 Forgot Password with OTP Verification using NodeMailer
🔒 Password Encryption using bcrypt
🎯 Session Management with Passport.js & Express Session
📢 Flash Messages for Success & Error Notifications
📊 Modern Dashboard with Statistics Cards
📂 Complete Category Management (CRUD)
📁 Complete Subcategory Management (CRUD)
🗂️ Complete Extra Category Management (CRUD)
🛍️ Complete Product Management (CRUD)
🖼️ Single & Multiple Image Upload using Multer
🔍 Search Functionality
📄 Pagination for Tables
↕️ Sorting (Newest, Oldest, Ascending, Descending)
✅ Active/Inactive Status Toggle
🔗 MongoDB Relationship Handling using populate()
✔️ Server-Side Form Validation
🚫 Duplicate Data Prevention
🗑️ Automatic Image Deletion on Record Removal
🧩 Reusable EJS Partials (Navbar, Sidebar, Footer)
📱 Fully Responsive UI for Desktop, Tablet & Mobile
